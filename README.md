# xor
Simple file XOR script.

## Usage
```
$ python xor.py -h
usage: xor.py [-h] --file FILE_PATH --key XOR_KEY [--suffix SUFFIX]

Save an XOR'd copy of a file.

optional arguments:
  -h, --help            show this help message and exit
  --file FILE_PATH, -f FILE_PATH
                        Path to the file to be XOR'd.
  --key XOR_KEY, -k XOR_KEY
                        XOR key to apply to the file.
  --suffix SUFFIX       Optional suffix for the output file.
```
